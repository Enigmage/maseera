# Girlfriend's Birthday Website

## Overview
An interactive, Bridgerton-inspired birthday website with a Victorian-era London aesthetic. Deployed via GitHub Pages as a static site.

## Tech Stack
- HTML/CSS/JavaScript (vanilla or lightweight framework)
- Static site deployable on GitHub Pages
- No backend required — all assets (photos, videos) stored in repo

## Project Structure
```
/assets/photos/     — uploaded photos
/assets/videos/     — uploaded videos
/assets/fonts/      — custom Victorian/serif fonts
/assets/audio/      — background music (optional)
/css/               — stylesheets
/js/                — scripts
/index.html         — main entry point
```

## Design Direction
- **Theme:** Bridgerton / Regency-era London
- **Color palette:** Soft golds, dusty roses, sage greens, cream/ivory backgrounds, deep navy accents
- **Typography:** Elegant serif fonts (e.g., Playfair Display, Cormorant Garant)
- **Textures:** Parchment, wax seals, floral motifs, ornate borders

## Key Features
- Envelope animations that reveal love notes/messages on click
- Photo carousel/slider galleries
- Media served directly from repo folders
- Mobile-responsive design
- Smooth scroll and page transitions

## Development
- Run locally with any static file server (e.g., `npx serve` or VS Code Live Server)
- Deploy by pushing to `main` branch with GitHub Pages configured

## Conventions
- Keep JS lightweight — avoid heavy frameworks for fast load on mobile
- Optimize images before committing (compress to reasonable sizes)
- Use semantic HTML and accessible markup where possible
